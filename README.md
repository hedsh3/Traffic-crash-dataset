README
Dataset Overview
This dataset was compiled as part of the research study:

"How a new metro line influences local traffic safety: A natural experiment in China"
(https://www.sciencedirect.com/science/article/pii/S0967070X26001253#sec5)

Description
The data included here was aggregated directly for use in the analysis presented in the above publication. All processing and aggregation steps were performed in accordance with the methods described in Section 5 of the article.
Upload of this dataset was for peer duplication of the original results

Key Points:
Origin: This dataset is a product of the study, as referenced above.
Processing: The raw data has been processed and aggregated for analytical purposes.
Usage: Data was used in the published research to investigate the impact of new metro line installation on local traffic safety.
